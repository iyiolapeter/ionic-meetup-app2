# ionic-meetup-app2

1. Install Dependencies

  ```bash
  npm i
  ```

2. Add Ios Platform

  ```bash
  ionic platform add ios
  ```

3. Build Ios Platform

  ```bash
  ionic build ios
  ```

4. Add Android Platform

  ```bash
  ionic platform add android
  ```

5. Build Android Platform

  ```bash
  ionic build android
  ```
